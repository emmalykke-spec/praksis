# ecoPraksis Design Workflow Rules

## Figma editing rules

1. **Always duplicate, never edit originals.**
   When making design changes, always copy the existing frame/component first.
   Rename the copy with a `[V2]` (or `[V3]`, etc.) prefix.
   Never modify a frame that already exists in the file.

2. **Always ask before implementing.**
   Before making any changes to the Figma file, ask the user:
   "Do you want me to implement this design change?"
   Wait for explicit confirmation before proceeding.
